### Option 1: Netlify (Recommended - Easiest)

**Method A: Drag & Drop (Fastest)**
1. Go to [netlify.com](https://www.netlify.com)
2. Sign up for a free account (or log in)
3. On the dashboard, find the "Sites" section
4. Drag and drop your entire project folder onto the deploy area
5. Your site will be live in seconds! You'll get a URL like `random-name-123.netlify.app`
6. (Optional) Click "Site settings" â†’ "Change site name" to customize your URL

**Method B: Git Integration (Recommended for updates)**
1. Create a GitHub repository and push your code
2. Go to [netlify.com](https://www.netlify.com) and sign up
3. Click "Add new site" â†’ "Import an existing project"
4. Connect your GitHub account
5. Select your repository
6. Netlify will auto-detect settings (no build command needed for static sites)
7. Click "Deploy site"
8. Every time you push to GitHub, your site will auto-update!

---

### Option 2: Vercel

1. Go to [vercel.com](https://vercel.com)
2. Sign up with GitHub, GitLab, or Bitbucket
3. Click "Add New Project"
4. Import your repository (or drag & drop your folder)
5. Vercel will auto-detect it's a static site
6. Click "Deploy"
7. Your site will be live with a URL like `your-project.vercel.app`

---

### Option 3: GitHub Pages

1. Create a GitHub account if you don't have one
2. Create a new repository (name it anything, e.g., "thunder-gaming")
3. Upload your files:
   - Click "uploading an existing file"
   - Drag all your files (index.html, assets folder, etc.)
   - Commit the changes
4. Go to repository Settings â†’ Pages
5. Under "Source", select "Deploy from a branch"
6. Choose "main" branch and "/ (root)" folder
7. Click "Save"
8. Your site will be live at `username.github.io/repository-name`

**Note:** It may take a few minutes for the site to appear.

---

### Option 4: Cloudflare Pages

1. Go to [pages.cloudflare.com](https://pages.cloudflare.com)
2. Sign up for a free account
3. Click "Create a project"
4. Connect your Git provider or upload directly
5. Select your repository or upload files
6. Click "Deploy site"
7. Your site will be live with a Cloudflare Pages URL

---

### Option 5: Surge.sh (Command Line)

1. Install Node.js from [nodejs.org](https://nodejs.org) if you don't have it
2. Open terminal/command prompt in your project folder
3. Run these commands:
   ```bash
   npm install -g surge
   surge
   ```
4. Follow the prompts:
   - Enter your email
   - Create a password
   - Choose a domain name (e.g., `thunder-gaming.surge.sh`)
5. Your site is live!
